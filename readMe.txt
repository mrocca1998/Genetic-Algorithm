There are eight states per generation. At each iteration, the two highest states and their evaluations are output.

Here are the instructions for running my project code.
1) Download the zip file submission
2) Unzip the file in the directory of your choosing
3) navigate to that directory
4) Place your input file(s) in the same directory as the .java files

5)
To compile the code enter the command "javac Genetic.java"

6) 
To run the algorithm enter the command "java Genetic filename"
With filename being the name of the input data file
